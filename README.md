# OverPing - online chat game
  ## install Backend package : 
      in ./BackEnd run 
       > npm install @nestjs/platform-socket.io socket.io 
       > npm fund 
  ## run Backend server : 
      npm run start:dev 
  ## install Frontend package : 
      in ./FrontEnd run  
        > npm install react-scripts 
        > npm fund 
  ## run Frontend server : 
      npm start
